## 配套项目

`vuedemo/` 是本教程的配套交互项目，包含所有章节的可运行示例。

```bash
# 启动交互式学习项目
npm run dev
# 打开 http://localhost:5173
```

## 学习资源汇总

### 官方文档（最权威）
- HTML/CSS/JS：https://developer.mozilla.org/zh-CN/
- Vue 3：https://cn.vuejs.org/
- Vue Router：https://router.vuejs.org/zh/
- Pinia：https://pinia.vuejs.org/zh/
- Vite：https://cn.vitejs.dev/

### 在线练习
- 浏览器直接运行 Vue：https://play.vuejs.org/
- 在线代码沙盒：https://stackblitz.com/
- CSS Flex 练习游戏：https://flexboxfroggy.com/#zh-cn
- CSS Grid 练习游戏：https://cssgridgarden.com/#zh-cn
- JS 练习平台：https://www.codewars.com/

### 视频教程
- 黑马程序员 Vue3（B站，免费）：搜索「黑马 Vue3」
- 尚硅谷 Vue3（B站，免费）：搜索「尚硅谷 Vue3」
- Vue Mastery（英文，部分免费）：https://www.vuemastery.com/

### 书籍推荐
| 书名 | 适合阶段 | 说明 |
|------|---------|------|
| 《JavaScript 高级程序设计》第4版 | 阶段1-2 | 红宝书，JS 圣经 |
| 《你不知道的 JavaScript》上中卷 | 阶段2 | 深入理解 JS 核心 |
| 《CSS 揭秘》 | 阶段1-2 | CSS 技巧大全 |
| 《深入浅出 Vue.js》 | 阶段3 | Vue 原理解析 |

### 社区
- 掘金：https://juejin.cn/（中文前端社区，质量高）
- 思否 SegmentFault：https://segmentfault.com/
- GitHub：https://github.com/（看源码、找项目）
